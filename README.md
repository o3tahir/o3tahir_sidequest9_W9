## Project Title

GBDA 302 Week 9 Side Quest: Debug Screen

---

## Authors

Karen Cochrane and David Han. Changes made to code by Oma Tahir using GenAI.

---

## Description

This project was based on GBDA 302 Week 9 Example 3. Using the provided code as a starting point, 
a debug screen was added to help with testing and development during gameplay. 
The debug screen allows the player to toggle moon gravity on and off, 
along with other helpful debugging features used to check how the game is working.

---

## Learning Goals

Learning Goals:

- Add a debug screen to an existing game
- Create toggles for testing gameplay features
- Use moon gravity as a switchable physics change
- Practice using debug tools during development
- Keep debug features separate from normal gameplay
